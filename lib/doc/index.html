API-as-a-Service Documentation

Users

Method URI Returns Description
GET /user Array Returns an array of all users
GET /user/<User ID> User Returns a user of the specified ID
POST /user User Request:
data: {
    email: email@test.com,
    fname: First name,
    lname: Last name,
    pass: passw0rd,
    active: 1
}
Returns the created User

Projects

Method URI Returns Description
GET /project Array Returns an array of all projects
GET /project/<Project Code> Project Returns a project of the specified code
POST /project Project Request:
data: {
    name: project name,
    code: PRJCD,
    active: 1
}
Returns the created Project
